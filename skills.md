# Skills Registry

The agents use these reusable skills through the LangChain prompt/tool layer.

| Skill | Purpose |
| --- | --- |
| `code_generation` | Generate complete runnable application code artifacts. |
| `test_generation` | Generate focused tests and concise test reports. |
| `code_review` | Review generated artifacts for bugs and maintainability risks. |
| `dockerfile_generation` | Generate Dockerfile, Compose, and local deployment instructions. |
